# HeatValve-6 ESP Dashboard

A lightweight, modular, framework-free dashboard optimized for ESPHome devices.

## Features
- Ultra-lightweight runtime
- Modular component system
- SSE real-time updates
- Offline mock mode
- SVG flow diagram with motor animation

## Structure
web/
  dashboard.js
  dashboard-src/
    core/
    utils/
    components/
    app/

## Usage
Include dashboard.js and mount to #app.

## Build
Use esbuild to bundle into a single file.
